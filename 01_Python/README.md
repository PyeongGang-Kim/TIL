터미널의 notes(til/01_Python/notes)에서 code ~/.bash_profile 입력

bash_profile 수정 창에

#Jupyter notebook alias

alias jp="python -m notebook"

입력 후 저장

안되면 source ~/.bash_profile



그러면 jp입력시 쥬피터노트북 켜짐.



터미널에서 code ~/.bashrc

뜨는 창에서

#Jupyter notebook alias

alias jp='python -m notebook'

안되면 source ~/.bashrc



저장 후 터미널 재실행하면 jp입력 시 쥬피터노트북 실행된다.
